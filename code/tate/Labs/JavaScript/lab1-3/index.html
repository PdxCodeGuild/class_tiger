Rot Cipher

This program can be used to encode an input. For each character, a corresponding character is found using the input number to "rotate" the character forward through the alphabet.

Credit Card Validator

The input number is checked to see whether it is a valid credit card number. (Not an actual credit card validation process). An example of a card number that returns 'Valid' with this program is 4556737586899855

Mock ATM system. The user may deposit or withdraw funds. The changes are reflected in the Account Balance. A transaction history is also provided.
